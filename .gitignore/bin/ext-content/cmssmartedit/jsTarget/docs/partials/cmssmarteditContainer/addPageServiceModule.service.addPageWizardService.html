addPageWizardService
service in module addPageServiceModule

Description

The add page wizard service allows opening a modal wizard to create a page.

Methods

  • getPageTypeFields(pageTypeCode)

    This method returns the list of generic editor fields used to provide all the page info necessary to create a page of a page type.

    Parameters
    ParamTypeDetails
    pageTypeCodeString

    The page type for which to retrieve its associated generic editor fields.

    Returns
    Array

    The list of generic editor fields associated to the provided page type. If no match is found, this method returns the set of default fields.

  • openAddPageWizard()

    When called, this method opens a modal window containing a wizard to create new pages.

    Returns
    Promise

    A promise that will resolve when the modal wizard is closed or reject if it's canceled.
