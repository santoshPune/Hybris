seMediaService
service in module seMediaServiceModule

Description

This service provides an interface to the $resource provided by the seMediaResource service and the seMediaResourceService service. It provides the functionality to upload images and to fetch images by code for a specific catalog-catalog version combination.

Methods

  • getMediaByCode(code)

    Fetches the media for the selected catalog corresponding to the specified code

    Parameters
    ParamTypeDetails
    codeString

    A unique code identifier that corresponds to the media as it exists in the backend.

    Returns
    Promise

    If request is successful, it returns a promise that resolves with the media POJO. If the request fails, it resolves with errors from the backend.

  • uploadMedia(media)

    Uploads the media to the catalog.

    Parameters
    ParamTypeDetails
    mediaObject

    The media to be uploaded

    PropertyTypeDetails
    codeString

    A unique code identifier for the media

    descriptionString

    A description of the media

    altTextString

    An alternate text to be shown for the media

    fileFile

    The File object to be uploaded.

    Returns
    Promise

    If request is successful, it returns a promise that resolves with the media POJO. If the request fails, it resolves with errors from the backend.
